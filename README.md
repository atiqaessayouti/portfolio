# Portfolio Web Personnel - AtiQa Essayouti

## Description du Projet
Ce projet consiste en la conception et le développement de mon portfolio web personnel. Réalisé dans le cadre d'un mini-projet, il a pour objectif de mettre en pratique les compétences acquises en développement front-end, notamment la structuration sémantique, la mise en page moderne et le design adaptatif.

Le site présente mon profil, mes compétences techniques, ainsi que les différents projets que j'ai réalisés au cours de ma formation.

## Fonctionnalités
- **Présentation** : Une section d'accueil introduisant mon profil de Développeuse Web & Mobile.
- **Compétences** : Une liste visuelle de mes compétences techniques (HTML, CSS, Java, Flutter, Firebase, Python, C++, PHP).
- **Projets** : Une galerie exposant mes principales réalisations avec des liens directs vers leurs dépôts GitHub respectifs.
- **Contact** : Les différents moyens de me joindre (Email, GitHub, LinkedIn).
- **Design Adaptatif (Responsive)** : Le site est conçu pour s'adapter automatiquement aux différentes tailles d'écran, y compris les appareils mobiles.

## Exigences Techniques Remplies
Ce portfolio respecte les contraintes techniques suivantes définies dans le cahier des charges :
- [cite_start]**Mise en page moderne** : L'utilisation de **Flexbox** a été privilégiée pour assurer une structure organisée, des alignements propres et des espacements harmonieux[cite: 5].
- [cite_start]**Design Cohérent** : Le site arbore une palette de couleurs sobres et cohérentes (2 à 3 couleurs principales maximum) avec une typographie lisible[cite: 6].
- [cite_start]**Responsive Design** : Le site est entièrement responsive et intègre au minimum une **media query** pour garantir un affichage optimal sur les terminaux mobiles[cite: 7, 11].

## Structure du Repository
- `index.html` : Fichier principal contenant le code HTML du portfolio.
- `style.css` : Fichier contenant l'ensemble des règles de style CSS, y compris les media queries pour le responsive design.
- `profile.jpg` : Image de profil utilisée dans la section d'accueil.
- `README.md` : Ce fichier, servant de rapport de projet et de documentation.

## Installation et Visualisation
Pour visualiser le projet localement :
1. Cloner le repository : `git clone https://github.com/atiqaessayouti/portfolio.git`
2. Naviguer dans le dossier du projet.
3. Ouvrir le fichier `index.html` dans un navigateur web.
